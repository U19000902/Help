3.14.3 Generate Sales Report

Purpose: This screen is used to display the sales report. The system will generate a sales report when accessed through the home screen. The system will calculate the gratuity earned per employee, the date the employee earned this money and the subtotal they earned through the month.

Item Num

Item Name

Description

          1.

       Save as PDF button

Click the “Save as PDF” button to export the sales report as a pdf format file.

 

        2.

        Add Condition button

Click the “Add Condition button” button to prompt and confirm the parameters that can be applied to the report data.

 

        3.

          And/Or Button

Click “And/Or “ Button to set the terms required for the set conditions to be true.

  4.

         Date dropdown input

Input and select a specific date condition to be applied to the report data.

  5.

         Start with dropdown input

Input and select a specific report entry to be the starting point as a condition to be applied to the report data.

6.

           Search bar

Search bar to help search for specific report entries.

         7.

          Back Button

Click Back Button to return to 3.14.1 Report Home Screen
