5.2 Restore Procedures

The system application is hosted on AWS as well in a separate instance. The backups of the application will be found on this connection and in Github. Should the administrators need to restore the system they can download previous versions through AWS or through Github. The system will confirm the restore of the system through the AWS Remote desktop connection.
