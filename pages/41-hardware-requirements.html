4.1 Hardware requirements

To run the system without delays, stutters or any other issues we recommend a laptop, desktop computer or tablet with the following requirements:

●     Stable internet connection

●     2GB Ram

●     200 MB HDD space

●     CPU @ 1.4 GHz
