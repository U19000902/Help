3.8.1 Product Home Screen

Purpose: To allow the users to view the product home screen and all its related functionalities.

 

No

Item Name

Item Description

1.

View Products button

“View Products” button to direct the user to the Product index page which is populated by all products.

2.

Add product button

“Add Product” button to allow user to add new products to system

3.

Import Products button

“Import Products” to allow user to import a Excel/csv file populated with new products

4.

Write-Off Products button

“Write-off Products” button to allow user to record write-off of products

5.

Complete stock take button

“Complete Stock Take” button to allow users to record Stocktakes

6.

Export Products button

“Export Products” button to allow users to export a pdf of all products on the system.
