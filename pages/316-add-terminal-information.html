3.1.6 Add Terminal Information

Purpose: To register and add POS Terminal devices to a merchant profile. The merchant will then be able to make sales with the terminal linked to their profile.

 

No.

Item Description

Description

1.

Serial Number textbox

This textbox will allow product owner to add a terminal serial number to the merchant profile that they will be making use of to perform sales.

2.

Add Terminal button

This button allows user to complete the addition of the terminal device to merchant.
