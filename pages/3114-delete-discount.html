3.11.4 Delete Discount

Purpose: This use case describes the process of a user permanently deleting the details of an existing discount and saving those changes on the system. The user selects the specific discount’s delete option and navigates to the “Delete discount Page” and confirms the deletion.

 

 

No

Item Name

Item Description

1.

Delete button

The user can click this button to begin the deletion of a discount.

2.

Confirm button

The user will be required to confirm the deletion of the discount.
