3.8.5 View Product Details

Purpose: The user will see a list of the product on the product view retrieved from the database. Each item will have a view button to view item details. The admin can click on this button to view more details about the item.

 

 

No

Item name

Item Description

1.

Edit button

The user can select this button to edit the current products details. The system will direct the user to the edit product screen.

2.

Delete button

The user can select the delete button to delete current product. The system will direct the user to the confirmation screen.

3.

Back button

The user can select this button to return to the product index screen.
