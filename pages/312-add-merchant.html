3.1.2 Add Merchant

Create Merchant screen

Purpose: The system should allow for the owner to add a merchant. The owner will input the new merchantsâ€™ details. The system will go through the database list of merchants to check for duplicates. When this check has been executed the system will prompt for confirmation and save the new merchant.

 

No.

Item Name

Description

1.             

Merchant Name textbox

Product owner can enter new Merchant Name. This field is required.

2.             

Email Address textbox

Product owner can enter merchants email address. This field is required.

3.             

Contact Number textbox

Enter new merchant cellphone number. This field is required.

4.             

Intecon Username textbox

Product owner can enter merchant intecon username that will be used in the linking of point of sale terminal devices. This field is required.

5.             

Intecon Password textbox

Password that will be used in the login with Intecon. This field is required.

6.             

Street Address textbox

Product owner can enter the street address of the Merchant. This field is required.

7.

Add Merchant button

The Product owner will click here to add the merchant.

8.

Back button

The product owner can use this to return to merchant home screen.
