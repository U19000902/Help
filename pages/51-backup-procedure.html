5.1 Backup Procedure

Our system is hosted on AWS. AWS has a very resilient system in terms of data loss and procedures. On AWS you can set the intervals for backups of the database. Our AWS environment will backup the database every 3 hours and should any issues arise with the database they can restore the data from a specific point.
