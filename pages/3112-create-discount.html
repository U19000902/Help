3.11.2 Create Discount

Purpose: This use case describes the process of a user entering the details of a new discount to be saved onto the system. The user navigates to the create discount page, enters the required details, and submits them. The system validates and stores them to the database.

 

 

No

Item Name

Item Description

1.

Description textbox

The user can enter a description for the discount. This field is required.

2.

Discount type dropdown list

The user can select the discount type that will be used in the amount textbox. If type “Percent” is chosen the amount would be the percentage and “Rands” will be actual discount amount.

3.

Amount textbox

The user can enter the amount of the discount.

4.

Start date picker

The user will be required to select a start date of when the discount will become active.

5.

End date picker

The user will be required to select a end date of the discount.

6.

Create discount button

The user clicks this button once they have completed entering the discount details.

7.

Back button

The user can use this button to return to the discount home screen.

8.

Add To Discount button

This button allows the user to add products from the system to the discount.

9.

Remove button

This button allows the user to remove a product from the discount.
