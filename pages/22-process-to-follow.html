2.2 Process to follow

When using the system the user may run intoÂ  a situation where they do not understand how to get help with the specific screen they are on. The user should click on the following icon that is present on the specific screen.

 

When the user clicks the icon the system will open up the user manual document on a hosted system with reference to the specific page that is being used. The user manual is hosted on Git Pages, and the references to the document will be based on the use case labelling we have done.
