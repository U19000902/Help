3.1.1 Merchant Navigation

Merchant Home screen

Purpose: To allow the Product owner to maintain merchants that will be making use of the system as well as managing VAT and Locations. You can sign out from the system, CRUD related functions are used in each merchant entry.

 

Num

Item Name

Description

1.            A

Add Merchant button

The product owner can add new merchants to the system.

2.             

Manage Location button

The product owner can add new locations.

3.             

Manage VAT Button

The product owner can update the vat value on the system.

4.             

Terminal button

The product owner can add a terminal payment device to merchant.

5.             

Edit button

To allow product owner to Update a merchants details.

6.             

View button

To allow product owner to view the details of merchants.

7.             

Archive button

To allow product owner to archive a merchant.

8.             

Sign-out button

To allow product owner to logout of the system.

9.

Search box

To allow product owner to search and filter the list of all merchants registered on the system.
