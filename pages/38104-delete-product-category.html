3.8.10.4 Delete Product Category

Purpose: The user would request to remove the product category from the system. The user clicks on the product category they would like to delete. The user confirms the deletion.

 

 

No

Item Name

Item Description

1.

Delete button

The user can click the delete button to remove a product category

2.

Confirm button

The user must click the confirm button to complete the deletion of a product category.
