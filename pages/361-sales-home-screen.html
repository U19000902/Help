3.6.1 Sales Home Screen

Purpose: To allow the user to either make sale or view a sale that has been made on the system.

 

Item Num

Item Name

Description

        1.

          Make a Sale Button

 Click the “Make a Sale” button

to go to the sale screen. Customer purchases can be done here.

 

 

        2.

          View Sales Button

Click the “View Sales” button

to go to the sale screen. You can view all past sales transactions here.
