3.12.5 Book Shift

Purpose: The user will select a shift, from there they will be directed to shift page where they will book a specific slot.

 

 

Item Num

Item Name

Description

          1.

          Edit Button

Click the “Edit” button to invoke 13.4 Update Shift.

 

        2.

         Search bar

Enter a query to search for specific slots

        3.

          View Button

Click “View“ Button the slot details

        4.

          Book  Button

Click “Book” Button to Book Shift in the selected slot.

         5.

          Back Button

Click Back Button to return to 3.12.1 Shift Home screen
