3.14.8 Generate Supplier Report

Purpose: The system will automatically generate a report with all the suppliers that is currently on the system. These reports will be made available on a “Supplier Report Page” where the user can click a button to view the report and another button to download a PDF of the report.

 

 

Item Num

Item Name

Description

          1.

       Total Approved Orders label

Displays the number of total approved orders data.

 

          2.

       Total Product Quantity label

Displays the total product quantity data.
